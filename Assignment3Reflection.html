  1. Provide a short description (in your own words) of assignment 3.

        

The dice game is a game where the main point is to roll dice. The first round, you must roll a two, seven, or eleven to win. If not, you move on to the next round. The second round, you must not roll a 7, 11, or 2 or or else you lose. You win if you match  your roll the previous round. It includes a menu and instructions.

  1. Describe any problems you faced while working on the assignment.

                I found it very difficult to create the formula for the dice. At the beginning, my game required the user to input the number on the image of the dice, making it very easy to cheat.

  1. How did you overcome them?

I created variables for each side of both die. I matched these variables to a value (1-6), so that the program inputs the variable instead of the user. This made it impossible for the user to cheat.
