ISP Unit

For my ISP, I will be creating a game inspired by the popular 2012 game: Grand Slam Tennis 2. It will feature a title screen where the game will start up on. You will be able to input your name for the scoring element, and select your difficulty based on your skill level. The game will run on a 2D platform and will be based on the game: Pong. It will include a score, and a screen indicating the winner at the end of the match. When the match is over, the user will be taken back to the main menu.
