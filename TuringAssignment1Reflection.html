Assignment Reflection

  1. Provide a short description (in your own words) of assignment 1

        I have written a program that will accept an input for: Purchase Price, Salvage Value, and Estimated life. The program will compute these values to find the yearly depreciation. It creates a list showing the year, the yearly depreciation, accumulated depreciation, and the book value to date. If 0’s are entered for every input, the program will put ”Program Terminated”. And will exit. The program does error checks for negative integers and string inputs. It also checks that the salvage value is less than the purchase price.

  1. Describe any problems you faced while working on the assignment.

                One of the biggest problems I faced was trying to figure out how to declare a variable as a real number, and convert it to a string value to allow for the error check. I also had trouble figuring out how to create the “Program Terminated” function.

  1. How did you overcome them?

                I relied heavily on the online turing wiki to search up the commands I would use to solve the problems mentioned in the previous question. I created a separate loop for the “Program Terminated” function using the exit command.

  1. What lessons have you learned for the future?

        Writing this program has definitely helped my understanding of error checks. I had to use problem solving to figure out how exactly I would get these error checks to work in the end which will always help me throughout my high school career and for everyday life.
