Assignment Reflection

  1. Provide a short description (in your own words) of assignment 2

                for assignment 2, we were to write a game program in Turing - Agarius. Agarius is derived from Agar.io, a popular online action game created by Matheus Valadares. Our program was centered around a yellow bubble controlled by the user. Our program had to include 7 different cookies represented by different colours. Each cookie has it’s own effect to the yellow bubble. The yellow bubble must stay within the borders of the run window, and it must not exceed a radius of more than 50 pixels or less than 2 pixels.

  1. Describe any problems you faced while working on the assignment.

                Figuring out how to make the yellow bubble eat the cookies was my greatest problem. It took my two days of trial and error to fix this problem. I also encountered trouble on the magenta cookie, where the direction of the arrow keys was supposed to flip (up is down, right is left, vise versa).

  1. How did you overcome them?

        I fixed the first problem by using a for loop. I created x and y coordinate variables for each cookie and applied the appropriate effect for each cookie using if statements. For the second problem, I created a speed variable (spd:=1) and multiplied it at the end of the arrow key statements. When the magenta cookie is eaten, the speed variable is multiplied by -1, effectively flipping the arrow key functions. I also used the speed variable for the orange and cyan cookie functions.

4.  What lessons have you learned for the future?

        This assignment has given me more experience using Turing graphics settings. I learned how to create key functions (arrow keys/letters) and how to make objects move. I feel that I could apply these newly acquired skills when creating games later on in the ICS course, and possibly even in other courses such as TEJ.
